# 100xDev Journey 🚀

Documenting my journey to becoming a world-class full-stack engineer through the 100xDevs curriculum and hands-on project building.

This repository contains notes, code examples, assignments, mini-projects, experiments, and learnings from every topic covered during the journey.

## Goal

- Build strong web development fundamentals
- Master JavaScript and TypeScript
- Become proficient in modern full-stack development
- Learn scalable backend architectures
- Understand real-world system design concepts
- Build production-ready applications
